# Awesome nsite
A collection of tools, apps and other stuff for using the nsite spec (NIP-5A) which extends the Blossom specification build upon the Nostr protocol.

## Protocol specs

- [NIP-5A nsite specification](https://github.com/nostr-protocol/nips/blob/master/5A.md)
- [Blossom specification](https://github.com/hzrd149/blossom)
- [nsite v2 "Pubkey Static websites" by hzrd149](https://github.com/hzrd149/nips/blob/cbe90dbcf7e492c7bc7bf04cd64c84598b435421/nsite.md)
- [nsite v1 "Static websites" by hzrd149](https://github.com/hzrd149/nips/commit/41e77b45a1e8a8d170097e363f7d7254797cc5c5)
- [Original nsite NIP draft by Lez](https://github.com/hzrd149/nips/commit/41e77b45a1e8a8d170097e363f7d7254797cc5c5)
- [Nostr NIPs](https://github.com/nostr-protocol/nips)

## Resources
- [nsite.run](https://nsite.run) Information and resources for nsite

## Tools

- [nsite-gateway](https://github.com/hzrd149/nsite-gateway) Typescript gateway for simple static websites on nostr
- [nsyte](https://github.com/sandwichfarm/nsyte) CLI deployment tool
- [nsite-action](https://github.com/sandwichfarm/nsite-action) A Github action that deploys nsites to nostr and blossom (using nsyte)
- [nsite debugger](https://nsite.info/debug) A tool to debug nsites.
